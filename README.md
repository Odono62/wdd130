# Odono John Jachukwu - Full-Stack Personal Website

A complete full-stack personal portfolio website with Node.js/Express backend, MongoDB database, authentication, and admin panel.

## Features

### Frontend
- Modern, responsive design with dark/light mode
- Animations and interactive elements
- Contact form with validation
- Admin authentication system
- Real-time project display

### Backend
- Node.js with Express.js
- MongoDB database with Mongoose
- JWT authentication
- RESTful API endpoints
- Email notifications
- Admin panel for contact management

## Tech Stack

### Frontend
- HTML5, CSS3, JavaScript
- Font Awesome icons
- Google Fonts (Poppins, Roboto)
- Animate.css for animations

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT for authentication
- bcryptjs for password hashing
- nodemailer for email
- CORS, Helmet, Rate Limiting
